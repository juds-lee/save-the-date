# Save the Date
Save the date is a special project I am working on as a 2025 bride. 
This project includes my user facing wedding invitation website, guest management tooling, authentication, and user verification.

Features: 
- Email template with unique rsvp links embedded
- Guest management dashboard 
- Website which users will be taken to upon clicking their email invite link. Website is responsive and dynamic to each user
- Authentication to ensure only those with the correct password or authorized links can access the website
- RSVP functionality for users to submit their attendance and other details
